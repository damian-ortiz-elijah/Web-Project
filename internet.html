What is Encryption?

Encryption is the process of converting data into a secure format to protect it from unauthorized access. It ensures confidentiality, integrity, and authenticity of data.

Symmetric Encryption

Uses a single key for both encryption and decryption. Common use cases include file encryption and secure communication.

Asymmetric/Symmetric Encryption

The difference from symmetric and asymmetric encryption is symmetric encryption uses the exact same key to decrypt and encrypt symmetric is usually faster but less secure asymmetric decryption and encryption uses a public key that will encrypt and decrypt well on the other end you have a private key that could only be used by one device that allows you to decrypt and encrypt this way sometimes takes a a lil longer built also provides more protection
